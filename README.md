## ✨ Features

- 🌍 **Real-time Weather** - Current conditions for any city
- 📅 **5-Day Forecast** - Detailed daily forecasts
- 🌡️ **Temperature Units** - Toggle between Celsius and Fahrenheit
- 🌓 **Dark/Light Theme** - Beautiful theme with persistence
- 💾 **Local Storage** - Saves theme and unit preferences
- 📱 **Fully Responsive** - Perfect on all devices
- 🎨 **Weather Icons** - Beautiful Lucide React icons
- ⚡ **Fast Loading** - Optimized API calls

## 🚀 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone git@github.com:abdullah-butt-dev/weather-app-react.git

# Navigate to project
cd weather-app-react

# Install dependencies
npm install

# Start development server
npm run dev
